# react-family


## 开发坏境启动

1. `npm install`
2. `npm start`
4. `npm run mock` 打开`json-server`,模拟`ajax`数据
3. 浏览器打开[http://localhost:8080](http://localhost:8080)

## 生产坏境部署

1. `npm run build`

2. 拷贝dist文件夹至服务器即可
